RipondaPhotodiodeGroup
-------------------------------

To import RipondaPhotodiodeGroup, you can either use:
```python
from psychopy_cedrus.riponda import RipondaPhotodiodeGroup
```
or, any time after `psychopy.plugins.activatePlugins` has been called:
```python
from psychopy.hardware.cedrus import RipondaPhotodiodeGroup
```
